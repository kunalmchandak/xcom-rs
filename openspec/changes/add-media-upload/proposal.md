# 提案: add-media-upload

## 目的
メディアのアップロード機能を追加し、投稿に画像/動画を添付できる前提を整える。

## 背景
投稿作成の実用性を上げるには、メディアIDを取得する手段が不可欠。

## 変更概要
- `media upload <path>` を追加する。
- `commands`/`schema`/`help` に新コマンドを反映する。

## 非スコープ
- チャンクアップロード
- メディアのメタデータ編集
- メディアの進捗ポーリング

## 成功条件
- `media upload` が `--output json|ndjson` で `media_id` を返す。
- ファイルパス不正時に明確なエラーが返る。

## 依存関係・リスク
- `POST /2/media/upload` と `media.write` スコープに依存。
- 大容量ファイルの取り扱いは次フェーズで扱う。
