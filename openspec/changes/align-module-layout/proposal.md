# align-module-layout Proposal

## 背景
`tweets` モジュールは `commands/models/ledger` に分割されていますが、他のモジュール（`auth`, `billing`）は同様のパターンに従っていません。

## 目的
モジュール構成を統一し、コードベース全体の一貫性を高めます。

## スコープ
- `auth`, `billing` モジュールの構成を `tweets` パターンに合わせる
- 既存の機能は維持

## 非スコープ
- 新しい機能追加
- モジュール間の依存関係の変更

## 成果物
- 統一されたモジュール構成
- リファクタリング後のドキュメント更新

## リスク/留意点
- 大規模な移動によるマージコンフリクトの可能性
