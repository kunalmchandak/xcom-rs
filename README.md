# xcom-rs

A CLI application written in Rust.

## Setup

### Prerequisites

- Rust (1.70+)
- [prek](https://github.com/j178/prek) (pre-commit hooks, auto-installed by setup script)

### Quick Start

1. Clone the repository:
```bash
git clone <repository-url>
cd xcom-rs
```

2. Run the setup script:
```bash
./.wt/setup
```

3. Build the project:
```bash
make build
```
